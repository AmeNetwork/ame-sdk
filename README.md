## ame-sdk
ame-sdk is a Javascript SDK used to interact with [Ame Network](https://ame.network/)

### Quick start
Install with npm:
```
npm i ame-sdk
```
Install with yarn:
```
yarn add ame-sdk
```
Install dependencies
```
npm i
```
Create an .env file in the root directory
```
PRIVATE_KEY=<Your Private Key>
```
Run test
```
yarn test
```
